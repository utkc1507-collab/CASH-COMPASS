# CashCompass

CashCompass is a simple full-stack expense tracking web project that demonstrates frontend-backend integration using REST APIs.

---

## 🚀 Features

- Add new expenses  
- View all expenses  
- Delete expenses  
- Search/filter expenses by category  
- Dynamic UI updates without page reload  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- API: REST API with JSON responses  

---

## ⚙️ How It Works

1. User enters expense details in the frontend form  
2. JavaScript sends data to backend using `fetch()`  
3. Flask API processes the request  
4. Data is stored in memory  
5. Backend returns JSON response  
6. Frontend updates UI dynamically

## 🔌 API Endpoints

| Method | Endpoint | Description |
|--------|---------|------------|
| GET | `/api/expenses` | Fetch all expenses |
| POST | `/api/expenses` | Add a new expense |
| DELETE | `/api/expenses/<id>` | Delete an expense |

---

## ⚠️ Note

Currently, the project uses in-memory storage for simplicity and to focus on API integration and frontend-backend communication. It can be extended to use a database like SQLite or MongoDB.
